medidas relativas:

porcentaje

em
Unidad relativa al tamaño de fuente especificada más cercana

rem
Unidad relativa al tamaño de fuente especificada en el ancestro mas lejano

forma de agregar una condicion para llamar una hoja de estilos
<link rel="stylesheet" href="css/media.css" media="screen and (max-width:768px)"/>

Dentro de CSS se declaran las condiciones:
@media screen and (max-width: 768px){
  body {
    background-color: pink;
  }
}

Y dentro del HTML entre las etiquetas style igual.

En el caso de hacer desktop first ir primero por los tamaños más grandes

1024px / 768px / 480px / 320px
